# 📊 Swiggy Data Analysis using SQL

## 🔍 Project Overview
This project analyzes restaurant data from Swiggy to extract meaningful business insights related to pricing, delivery performance, and customer satisfaction.

---

## 🛠 Tools Used
- MySQL
- SQL
- GitHub

---

## 📂 Project Structure
- `queries.sql` → SQL queries for analysis
- `insights.md` → Key business insights derived from the data

---

## 📈 Key Insights
- Premium restaurants have higher ratings
- Faster delivery leads to better customer satisfaction
- Delivery efficiency impacts overall performance

- Analyzing real-world datasets
- Extracting business insights from data
